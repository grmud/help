Basic Structures

While there are many more structures that can be built, this list is provided to the new players who need to learn what structures are necessary for maintaining an extremely basic lair. All these structures can be constructed by a builder with the basic structure special ability.

Note: Many structures that different names, dependent upon the caste of the builder, but function the same. If the names vary for the structures listed here, the caste names are given for that structure.
