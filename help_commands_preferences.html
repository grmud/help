Summary: Commands: Preferences


Usage: pref?erences

By typing preferences you can customize some of the basic text colors shown in the window of your browser.

  • Colorize Grumble Output -- This option toggles the option to color blue, information coming from your grumble.
  • Colorize Action Target Output -- This option toggles the option to color red, all attacks made against you.
  • Set Room Description Highlight Colors -- This option lets you choose whether to display all available exits as blue or gray.
