Bulder Commands

You can command grunts to do a number of actions, however, all grunt commands must be preceded by:
command <grunt> to <action>
(e.g. command grunt to stop, or command grunt to follow me)
Here is a list of actions that builders can command grunts to do.
  • build <structure> to the <direction> -- The grunt will begin constructing a structure in the specified direction. (Note: you cannot order a grunt to build a lair entrance. To build an entrance to a new lair, you must first 'found your lair.'
  • portal opening <direction> -- The grunt will construct a portal in a specified direction, which creates an opening between two structures that have already been built.
  • destroy <here or direction> -- The grunt will start tearing down the room. (Note: if you or the grunt are standing within the room when it collapses, you and the grunts will die.)
  • repair -- The grunt will start repairing the structure at its current location.
  • mine -- If a grunt is in a mine, it starts gathering resources. Once it has filled up on resources it will deposit them in your storage chamber(see Mining for more information)
  • assist <direction> -- Grunt will assist another grunt that is currently building a structure in a lair, or he will continue to construction a on an unfinished structure. You can have up to four grunts constructing a structure.
  • upgrade -- That grunt will begin upgrading his current location if favors have been invested in the location by using the favor command.
