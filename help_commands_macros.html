Summary: Commands: Macros


Macros

A macro is a helpful shortcut you can use in place of typing full commands or emotes. In Grendel's Revenge there are two ways you can use macros.

The first method uses the @MACRO command. Macros of this type are best suited for simple and repetative actions like approaching, attacking, or emoting. When using this type of macro, remember that it is easier to use a shorthand name that you would not use during normal gameplay.

Grendel's Revenge Macros

The second method of using macros in Grendel's Revenge uses the built in capabilities of the Alice client. These macros are much more powerful and flexible than the @MACRO. Follow this link to learn about these macros.
