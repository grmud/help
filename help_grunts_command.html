Commanding Grunts

You can command grunts to do a number of actions, however, all grunt commands must be preceded by:

command <grunt> to <action>
(e.g. command grunt to stop, or command grunt to follow me)

Attempting to command a grunt will first look for the grunts that you control first, rather than the normal matching order.

Here is a list of actions that all monsters can command grunts to do.

  • follow <me or character> -- Grunt will follow you or the character you specify it to follow.
  • invite <grunt> -- Invites the grunt to serve under you. However, your role limits the number of grunts that you can command, and if the grunt is already under the service of someone else, you cannot invite it.
  • discharge <grunt> -- Discharges the grunt from your service, and makes the grunt available for others to command.
  • rename <grunt> -- Sets the grunt's name
  • stop -- Grunt will stop whatever it is doing.
  • Inventory Management
    • inventory -- Shows what the grunt is carrying.
    • get <item> -- The grunt will pick up the specified object
    • get <item> from corpse -- The grunt will take an item off a corpse
    • get <amount> <resource> from corpse or pile of loot -- The grunt will get the specified amount of resources from the specified location.
    • drop <item> or <amount> <resource> -- The grunt will drop the specified object or resources.
    • wear <item> -- The grunt will wear the item specified if it has the item in its inventory.
  • Harpy Grunts -- Only the chimera grunts can perform these maneuvers.
    • fly -- The grunt will start flying
    • land -- The grunt will land
