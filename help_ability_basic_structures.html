Summary: Abilities: Builder Special Abilities: Basic Structures


 Basic Structures

This is the basic ability for builders. This ability allows builders to construct the basic structures which are needed to build and maintain a lair. As the builder advances in this skill, the structures built will increase in quality and durability.

Command: N/A

This ability is based on:

  • Agility
  • Energy
