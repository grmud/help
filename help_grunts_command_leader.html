Leader and Warrior Commands

You can command grunts to do a number of actions, however, all grunt commands must be preceded by:
command <grunt> to <action>
(e.g. command grunt to stop, or command grunt to follow me)
Here is a list of actions that leaders and warriors can command grunts to do.
  • attack <character> -- The grunt will attack the specified character if his morale his high enough.
  • (Leader Only) guard <character|item|portal> -- The grunt will guard the object from attacks.
  • Patrolling -- You can command a grunt to patrol a route with these commands
    • route <start> -- Sets the beginning of a route for the grunt to follow.
    • route <store> <name> -- Ends the route and stores it as the specified name.
    • route list -- Shows a list of routes the grunt knows.
    • route clear <name> -- Clears the route from the grunt's memory.
    • patrol <name> -- The grunt will patrol the specified route.
