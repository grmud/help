Alice Client Macros

This method of using macros in Grendel's Revenge uses the built in capabilities of the Alice client. These macros are more powerful and flexible than those made with @MACRO.

There are five text sensitive (they must be in ALL CAPS) commands you can use:

  • MACRO ADD - Creates a new Alice macro. For example:

    MACRO ADD cf claw fighter

    Using the % symbol and a number allows you to use variables as part of your macro, where you can substitute npcs or objects in your environment. You can theoretically create an Alice macro with as many targets as you wish.

    MACRO ADD c claw %1

    Then you would type: c fighter, to attack a fighter, or c wizard to claw a wizard.

    The one thing you must always consider when creating a new Alice macro is the name must not be anything that is a word or command that can already be used. A simple example would be to never use 'a' as a macro name. Anytime you type a single 'a' the macro would take it's place.

  • MACRO STORE - Stores the last Alice macro you created. You must always use this after MACRO ADD or else the macro will not be there when you call on it again.

  • MACRO LIST - Lists all Alice macros on the computer system you are currently using.

  • MACRO REMOVE - Removes the Alice macro you name.

  • MACRO REMOVEALL - Removes all Alice macros on that computer system.

    If you would like to read more about the Alice client and Alice macros, go to http://www.skotos.net/clientdist/alice.html for more information.
